Mis mejores habilidades

Algunos de mis proyectos recientes

Crypto App

Aplicación Android con la visualización de un listado de criptomonedas y la información detallada de cada una. La app de criptomonedas se basa en la arquitectura MVVM y utiliza el patrón Repository. Se comunica con una base de datos como fuente principal de datos y emplea Retrofit para interactuar con la API (docs.coincap.io). App desarrollada en Kotlin.

EHA! Entrenamiento de Habilidades Auditivas

La aplicación EHA para Android, enfocada en mejorar habilidades auditivas, se desarrolló con Java y sigue el patrón MVP. Utiliza Firebase para autenticación y almacenamiento de datos, junto con MediaPlayer y Glide. La interfaz sigue el diseño Material. El proyecto se gestiona con SCRUM y está en curso en su versión 1.0.1.

Explorador de Productos

La app de explorador de productos sigue una arquitectura MVVM y hace uso del patrón Repository. Se conecta a una API para obtener categorías de productos, listar productos por categoría y mostrar detalles del producto. La app se desarrolla en Kotlin y permite a los usuarios ingresar evaluaciones y comentarios sobre los productos, que se almacenan en una base de datos Room.
